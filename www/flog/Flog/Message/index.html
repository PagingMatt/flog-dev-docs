Module Flog.Message

Message module encapsulates types and a constructor needed for log messages.

type level =
| Trace
| Debug
| Information
| Warning
| Error
| Fatal

The levels at which messages are logged.

type t

Message type.

val make : Unix.tm option -> level -> string -> t

Constructor for the Message.t type. If no time is passed in then the current UTC time when called will be stamped onto the message.

returns

The message constructed of the parameters passed in.

val cmp_tm : t -> t -> int

Compares two messages based on the time they were logged at. Ensures that more recent messages appear first.

returns

A negative int if a is greater than b, 0 if they are equal and a positive int if a is less than b.

Helper function to serialize messages.

returns
